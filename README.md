# MoodWav: A Mood Analyzer for Spotify

MoodWav is a NextJS application that uses Spotify's Web API to fetch a user's recently played tracks, calculates the mood of the user based on these tracks, and displays the data in a user-friendly interface. The user can filter the recently played tracks based on the mood, as well as download an image of their mood score.

## Features

- Fetches a user's recently played tracks from Spotify
- Calculates the mood of the user based on these tracks
- Displays the mood data in a `MoodScoreCard` component
- Displays the recently played tracks in a `RecentlyPlayed` component
- Allows the user to filter the recently played tracks based on the mood

## Getting Started

To get started with MoodWav, you'll need to clone the repository and install the dependencies:

```bash
git clone https://github.com/seangray-dev/moodwav.git
cd moodwav
npm install
```
